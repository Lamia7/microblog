# Microblog using Flask

## Créer répertoire

```bash
$ mkdir microblog
$ cd microblog
```

## Créer environnement virtuel et l'activer depuis le répertoire

```bash
$ python3.7 -m venv venv
```

```bash
$ source venv/bin/activate
(venv) microblog$ _
```

## Installer flask

```bash
microblog$ pip3 install flask
```
